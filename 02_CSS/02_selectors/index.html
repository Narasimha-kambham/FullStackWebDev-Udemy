CSS Selectors

A CSS selector selects the HTML element(s) you want to style.

Some of the commonly used selectors are as mentioned below:

  • Universal Selector
  • Element Selector
  • Group Selector
  • Id Selector
  • Class Selector
  • Child Selector
  • Descendant Selector
  • Pseudo-Class Selector
  • Pseudo-Element Selector
